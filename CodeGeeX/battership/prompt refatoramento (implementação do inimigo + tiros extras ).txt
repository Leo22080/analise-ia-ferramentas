Quero que você refatore o script.js para implementar duas melhorias:

Inteligência do inimigo:

Quando acertar uma parte de um navio, deve tentar as quatro posições adjacentes (cima, baixo, esquerda, direita).

Tiros extras para ambos os jogadores:

Se o player ou o inimigo acertar um navio, ganha direito a jogar novamente.

O turno só passa para o adversário quando o tiro for na água.
